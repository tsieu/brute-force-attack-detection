# **Brute Force Attack Detection Project**

## **📌 Overview**

This project demonstrates my ability to detect and analyze brute force attacks within a SOC environment. The focus is on identifying repeated failed login attempts, recognizing attack patterns, and documenting findings.

## **🎯 Objectives**

* Identify brute force attack patterns  
* Detect repeated failed login attempts  
* Differentiate between normal and malicious behavior  
* Document findings and response actions

## **🧠 Skills Demonstrated**

* Log analysis and investigation  
* Threat detection and pattern recognition  
* SOC analysis workflows  
* Incident documentation and reporting

## **🛠️ Tools Used**

* Log analysis tools

## **🔑 Key Concepts**

* Brute force attack patterns  
* Failed login attempt thresholds  
* IP reputation and anomaly detection

## **🚀 Outcome**

Successfully identified brute force attack activity through log analysis and documented the investigation process, demonstrating SOC-level detection and response skills.
